Setup the App

Lets start at the very beginning. It's a very good place to start

I recommend you use vs code for this project and install the 'vue VS Code Extension Pack' which will set you up with everything you need.

Next open prettier - code/preferences/settings and search for prettier and make sure that it’s setup how you require it

Once we have all this setup we are ready to install nuxt so open a terminal window. Make sure you are creating it in the desired place so cd myproject for example. When you create a nuxt project it will create a folder for you so keep that in mind when running this command

npx create-nuxt-app project-name

Once you run the command you will get the following and one by one in the command you need ot answer the questions. Here is what I reccommend just change the name and description to whatever project you are working on.

Once everything is setup you will see the following:

Personally I prefer to open Code again rather than cd into the project as then I am in the right workspace from the start and don't have to think about cd into a folder. However if you prefer to not open the project from the start then just cd into the folder like above chaning learn-nuxt-app for what you called your project. Then if you run the dev command you should see your project :)

You should now get this lovely message from Nuxt telling you where your project us. Use command and click to open the projet or just copy the route and paste it into your browser

Oops you may notice that you have lint errors. In order to fix them add this to the scripts of your package.json just under the lint script that is already there

"lint:fix": "eslint --fix --ext .js,.vue --ignore-path .gitignore .”

Then you can run:

npm run lint:fix

Now it's looking good, right!!
