webpack Configuration

Lets get webpack configured

webpack Analyzer

Nuxt.js use webpack-bundle-analyzer to let you visualize your bundles and how to optimize them.

Go to your nuxt-config and add the following

build: {
    analyze: true,
    }

Then run the analyzer and get analyzing

nuxt build -a

Extract Css

To extract your css into a separate file add the following to your nuxt.config

build: {
    extractCSS: true,
    }
